# streak-tracker

## Languages and Tools:
- Java
- JavaScript
- Tailwind CSS
- Spring Boot
- React
- MongoDB
- Maven

## Mission:
- Supports the user with maintaining their habits by utilizing a streak system, helping the user visualize their progress, and share a sense of pride of maintaining a high streak.

## Features:
- Intuitive web app that lets users create a habit, giving it a name and description, and lets the user increment the streak daily after completion. Automatic streak reset down to 0 if the user does not increment the streak within the calender day.

## Adding next:
- Email reminder for the user if the habit is still not completed by a certain time within the day.
- Authentication to allow for multiple users with personal habits
- Leaderboard of streaks for all users to inspire friendly competition and to hold each other accountable in their journey
- Graph of completion time for the user to visualize habit completion
