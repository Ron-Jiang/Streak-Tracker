# Streak-Tracker 
Java, JavaScript, Tailwind CSS
<br> Spring Boot, React, MongoDB, Maven
<br> OAuth 2.0

## Mission:
Supports the user with maintaining their habits by utilizing a streak system, helping the user visualize their progress, and share a sense of pride in maintaining a high streak.

## Features:
Intuitive web app that uses OAuth 2.0 to allow users to use their GitHub to sign-in. Once authenticated, it allows users to create a habit by giving it a name and description, and lets the user increment the streak daily after completion. Automatic streak reset down to 0 if the user does not increment the streak within the calender day.

## Future Features:
- Leaderboard of streaks for all users to inspire friendly competition and to hold each other accountable
- Email reminder for the user if the habit is still not completed by a certain time within the day.
- Graph of completion time for the user to visualize habit completion
